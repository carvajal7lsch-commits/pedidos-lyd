Ficha Técnica

Especificaciones técnicas y funcionales del ecosistema logístico de distribución Pedidos LYD.

Arquitectura MVC

Estructura monolítica optimizada para alto rendimiento, implementada sin dependencias de frameworks externos para garantizar control total y mínima huella de memoria.

  • Clean Logic
  • Modular
  • Scalable

Core & Data

Backend robusto basado en PHP 8.2 con una capa de persistencia en MariaDB.

  • PHP 8.2 (Strict Types)
  • MySQL / MariaDB
  • MySQLi (Native)
  • Secure Auth

PWA Native

Experiencia de aplicación instalable con manifest y soporte para pantalla completa en móviles.

GPS & Mapas en Tiempo Real

Integración profunda con Leaflet.js para el seguimiento en vivo de la flota de vendedores, permitiendo auditoría geográfica inmediata desde el panel central.

Security Core

Protección contra SQL Injection mediante Sentencias Preparadas en todo el sistema.

Ecosistema de Módulos

ADMIN

Dashboard Bento, Gestión de Carteras, KPIs de Ventas, Control de Inventario Global.

VENDEDOR
